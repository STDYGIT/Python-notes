## Unit I: Python Fundamentals

### 1. Python Data Structures

Python provides built-in data structures to store and organize data efficiently.

**Basic Data Types:**
- **int**: Integer numbers (e.g., 42, -10)
- **float**: Decimal numbers (e.g., 3.14, -0.5)
- **str**: Text strings (e.g., "Hello")
- **bool**: Boolean values (True, False)

```python
# Examples
age = 25                    # int
price = 19.99              # float
name = "Alice"             # str
is_student = True          # bool
```

### 2. Control Statements

Control statements determine the flow of program execution.

#### If-Else Statement

The if-else statement executes code based on conditions.

```python
# Basic if-else
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# Output: You are an adult
```

**Working:** Python evaluates the condition. If True, the if block executes; otherwise, the else block runs.

#### Nested-If Statement

If statements can be placed inside other if statements for complex decision-making.

```python
marks = 85
if marks >= 50:
    if marks >= 75:
        print("Distinction")
    else:
        print("First Class")
else:
    print("Fail")

# Output: Distinction
```

#### Elif Statement

The elif (else-if) statement checks multiple conditions sequentially.

```python
score = 75

if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
else:
    grade = 'F'

print(f"Grade: {grade}")
# Output: Grade: C
```

### 3. Expression Evaluation & Float Representation

**Expression Evaluation:** Python follows operator precedence (PEMDAS).

```python
result = 10 + 5 * 2        # Multiplication first
print(result)              # Output: 20

result = (10 + 5) * 2      # Parentheses first
print(result)              # Output: 30
```

**Float Representation:** Floating-point numbers can have precision limitations.

```python
x = 0.1 + 0.2
print(x)                   # Output: 0.30000000000000004

# Use round() for precision
print(round(x, 2))         # Output: 0.3
```

### 4. Loops

Loops allow repeated execution of code blocks.

#### While Loop

Executes as long as the condition remains True.

```python
count = 1
while count <= 5:
    print(f"Count: {count}")
    count += 1

# Output: Count: 1, Count: 2, ..., Count: 5
```

#### For Loop

Iterates over sequences (lists, strings, ranges).

```python
# Iterate over range
for i in range(1, 6):
    print(i)

# Iterate over list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
```

#### Nested Loops

Loops inside other loops for multi-dimensional iteration.

```python
# Multiplication table
for i in range(1, 4):
    for j in range(1, 4):
        print(f"{i} x {j} = {i*j}")
```

#### Break and Continue

- **break**: Exits the loop immediately
- **continue**: Skips the current iteration

```python
# Break example
for i in range(10):
    if i == 5:
        break
    print(i)  # Prints 0 to 4

# Continue example
for i in range(5):
    if i == 2:
        continue
    print(i)  # Prints 0, 1, 3, 4 (skips 2)
```

### 5. Functions

Functions are reusable blocks of code that perform specific tasks.

#### Parts of a Function

```python
def greet(name):           # Function definition
    """Greets a person"""  # Docstring (optional)
    message = f"Hello, {name}!"
    return message         # Return statement

# Function call
result = greet("Alice")
print(result)              # Output: Hello, Alice!
```

**Components:**
- **def**: Keyword to define a function
- **Function name**: Identifier for the function
- **Parameters**: Input values (optional)
- **Body**: Indented code block
- **Return**: Output value (optional)

#### Execution of a Function

When called, Python:
1. Passes arguments to parameters
2. Executes the function body
3. Returns control to the caller with the return value

```python
def add(a, b):
    return a + b

result = add(5, 3)  # Function call
print(result)       # Output: 8
```

#### Keyword and Default Arguments

**Default Arguments:** Provide default values for parameters.

```python
def power(base, exponent=2):
    return base ** exponent

print(power(5))        # Output: 25 (uses default exponent=2)
print(power(5, 3))     # Output: 125
```

**Keyword Arguments:** Specify arguments by parameter name.

```python
def introduce(name, age, city):
    print(f"{name}, {age} years old, from {city}")

introduce(city="Mumbai", name="Raj", age=25)
# Output: Raj, 25 years old, from Mumbai
```

#### Scope Rules

**LEGB Rule:** Local â†’ Enclosing â†’ Global â†’ Built-in

```python
x = "global"           # Global scope

def outer():
    x = "enclosing"    # Enclosing scope
    
    def inner():
        x = "local"    # Local scope
        print(x)
    
    inner()
    print(x)

outer()
print(x)

# Output:
# local
# enclosing
# global
```

### 6. Strings

Strings are sequences of characters enclosed in quotes.

#### Length and Operations

```python
text = "Python"
print(len(text))           # Output: 6 (length)

# Concatenation
greeting = "Hello" + " " + "World"
print(greeting)            # Output: Hello World

# Repetition
echo = "Ha" * 3
print(echo)                # Output: HaHaHa
```

#### Indexing

Access individual characters using index positions (starting from 0).

```python
text = "Python"
print(text[0])             # Output: P (first character)
print(text[-1])            # Output: n (last character)
print(text[-2])            # Output: o (second from end)
```

#### Slicing

Extract substrings using `[start:end:step]`.

```python
text = "Python Programming"

print(text[0:6])           # Output: Python
print(text[7:])            # Output: Programming
print(text[:6])            # Output: Python
print(text[::2])           # Output: Pto rgamn (every 2nd char)
print(text[::-1])          # Output: gnimmargorP nohtyP (reverse)
```

## Unit II: Advanced Python Concepts

### 1. Tuples

Tuples are immutable sequences enclosed in parentheses.

```python
# Creating tuples
coordinates = (10, 20)
colors = ("red", "green", "blue")

# Accessing elements
print(coordinates[0])      # Output: 10

# Tuples are immutable
# coordinates[0] = 15      # Error: cannot modify
```

**Use Cases:** Fixed data, dictionary keys, multiple return values.

---

### 2. Unpacking Sequences

Extract values from sequences into variables.

```python
# Tuple unpacking
point = (5, 10)
x, y = point
print(x, y)                # Output: 5 10

# List unpacking
numbers = [1, 2, 3, 4, 5]
first, *middle, last = numbers
print(first)               # Output: 1
print(middle)              # Output: [2, 3, 4]
print(last)                # Output: 5
```

---

### 3. Lists

Lists are mutable sequences enclosed in square brackets.

```python
# Creating lists
fruits = ["apple", "banana", "cherry"]

# Accessing elements
print(fruits[1])           # Output: banana

# Modifying lists
fruits.append("orange")    # Add to end
fruits.insert(1, "mango")  # Insert at index
fruits.remove("banana")    # Remove by value
fruits.pop()               # Remove last item

print(fruits)              # Output: ['apple', 'mango', 'cherry']
```

---

### 4. Mutable Sequences

Lists are mutable; tuples and strings are not.

```python
# List (mutable)
numbers = [1, 2, 3]
numbers[0] = 10
print(numbers)             # Output: [10, 2, 3]

# String (immutable)
text = "hello"
# text[0] = "H"            # Error: strings are immutable
text = text.capitalize()   # Create new string
print(text)                # Output: Hello
```

---

### 5. List Comprehension

Concise way to create lists using a single line of code.

```python
# Traditional approach
squares = []
for i in range(1, 6):
    squares.append(i ** 2)

# List comprehension
squares = [i ** 2 for i in range(1, 6)]
print(squares)             # Output: [1, 4, 9, 16, 25]

# With condition
evens = [x for x in range(10) if x % 2 == 0]
print(evens)               # Output: [0, 2, 4, 6, 8]
```

### 6. Sets

Sets are unordered collections of unique elements.

```python
# Creating sets
fruits = {"apple", "banana", "cherry"}
numbers = {1, 2, 3, 3, 4}  # Duplicates removed
print(numbers)             # Output: {1, 2, 3, 4}

# Set operations
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)               # Union: {1, 2, 3, 4, 5, 6}
print(a & b)               # Intersection: {3, 4}
print(a - b)               # Difference: {1, 2}
```

### 7. Dictionaries

Dictionaries store key-value pairs.

```python
# Creating dictionaries
student = {
    "name": "Alice",
    "age": 20,
    "grade": "A"
}

# Accessing values
print(student["name"])     # Output: Alice
print(student.get("age"))  # Output: 20

student["age"] = 21        # Update value
student["city"] = "Delhi"  # Add new key-value

# Iterating
for key, value in student.items():

```

### 8. Higher Order Functions

Functions that take other functions as arguments or return functions.

#### Functions as First-Class Objects

```python
def square(x):
    return x ** 2

def apply_function(func, value):
    return func(value)

result = apply_function(square, 5)
print(result)              # Output: 25

# Assigning functions to variables
operation = square
print(operation(4))        # Output: 16
```

#### Lambda Expressions

Anonymous functions defined using the `lambda` keyword.

```python
# Regular function
def add(x, y):
    return x + y

# Lambda equivalent
add_lambda = lambda x, y: x + y
print(add_lambda(3, 5))    # Output: 8

# With higher-order functions
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x ** 2, numbers))
print(squared)             # Output: [1, 4, 9, 16, 25]

# Filter example
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)               # Output: [2, 4]
```

### 9. Exception Handling

Handle runtime errors gracefully using try-except blocks.

```python
# Basic exception handling
try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"Result: {result}")
except ZeroDivisionError:
    print("Error: Cannot divide by zero")
except ValueError:
    print("Error: Invalid input")
finally:
    print("Execution completed")

# Raising exceptions
def validate_age(age):
    if age < 0:
        raise ValueError("Age cannot be negative")
    return age
```

### 10. User-Defined Modules and Packages

#### Modules

A module is a Python file containing functions and variables.

**math_operations.py:**
```python
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

PI = 3.14159
```

**main.py:**
```python
import math_operations

result = math_operations.add(5, 3)
print(result)              # Output: 8
print(math_operations.PI)  # Output: 3.14159

# Alternative import
from math_operations import multiply
print(multiply(4, 5))      # Output: 20
```

#### Packages

A package is a directory containing multiple modules and an `__init__.py` file.

**Directory Structure:**
```
mypackage/
    __init__.py
    module1.py
    module2.py
```

**Usage:**
```python
from mypackage import module1
from mypackage.module2 import some_function
```

### 11. File Handling

Reading from and writing to files.

#### Reading Files

```python
# Read entire file
with open("data.txt", "r") as file:
    content = file.read()
    print(content)

# Read line by line
with open("data.txt", "r") as file:
    for line in file:
        print(line.strip())
```

#### Writing Files

```python
# Write to file (overwrites)
with open("output.txt", "w") as file:
    file.write("Hello, World!\n")
    file.write("Python File Handling")

# Append to file
with open("output.txt", "a") as file:
    file.write("\nAppended text")
```

#### File Modes

- **"r"**: Read (default)
- **"w"**: Write (overwrites)
- **"a"**: Append
- **"r+"**: Read and write
- **"b"**: Binary mode (e.g., "rb", "wb")

```python
# Working with CSV files
import csv

# Writing CSV
data = [["Name", "Age"], ["Alice", 25], ["Bob", 30]]
with open("data.csv", "w", newline="") as file:
    writer = csv.writer(file)
    writer.writerows(data)

# Reading CSV
with open("data.csv", "r") as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)
```
